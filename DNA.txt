## 1. الفكرة الجوهرية

### المشكلة التي يحلها التطبيق
**من جهة الزبون:**
- يدخل محل ملابس ولا يعرف ما يناسب بنيته وطوله وميزانيته
- يتردد طويلاً ثم يخرج بدون شراء رغم رغبته
- لا يعرف كيف ينسق القطع مع بعضها بذكاء
- 80% من الملابس تُنسى بعد ارتدائها مرة واحدة

**من جهة البائع:**
- قطع مكدسة لا تُباع رغم جودتها
- لا يعرف لماذا يهرب الزبون
- يعتمد على الحدس في قرارات الشراء من الجملة
- لا يملك أي أداة رقمية للتحليل

### الجمهور المستهدف
- **أولاً:** بائعو الملابس الرجالية في ورقلة والجنوب الجزائري
- **ثانياً:** الشباب 18-35 سنة — زبائن المحلات
- **مستقبلاً:** عموم الجزائر ثم المغرب العربي

### ما يجعله مختلفاً
- لا منافس محلي واحد في الجزائر
- يعمل بالمخزون الحقيقي للمتجر — لا اقتراحات وهمية
- ذكاء اصطناعي حقيقي (Anthropic Claude API)

## 4. قرارات التصميم والـ UI/UX

### الألوان الثابتة (لا تتغير أبداً)
```css
--bg:       #0B0A08   /* أسود دافئ */
--gold:     #D2AF69   /* ذهبي رئيسي */
--goldD:    #8A7040   /* ذهبي داكن */
--goldL:    #EDD080   /* ذهبي فاتح */
--cream:    #EDE3CC   /* كريمي */
--rust:     #A83D22   /* أحمر صدئ — CTA */
--rustL:    #C94E2A   /* أحمر فاتح */
--sage:     #4A7848   /* أخضر زيتي */
--card:     #1C1915   /* بطاقات */
--txt:      #D8CFBA   /* نص عام */
--txtD:     #7A7060   /* نص خافت */
```

### الخطوط
- **Fraunces** (serif italic) — العناوين والأرقام الكبيرة — يعطي طابع الفخامة
- **Tajawal** — النصوص العربية والأزرار

### الطابع البصري
- فخامة هادئة — ليس صاخباً
- أسود دافئ + ذهبي = راقٍ لا باهت
- Glassmorphism خفيف على البطاقات
- أنيميشن بطيء ناعم — cubic-bezier springs
- لا ألوان زاهية — كل شيء مكتوم

### قرارات لا تتغير
1. البساطة في الواجهة — التعقيد في الخوارزمية
2. شاشات منفصلة لا صفحة واحدة طويلة
3. كل أنيميشن يخدم القصة لا العكس
4. Demo Data للعرض — البائع يضع بياناته لاحقاً

## 8. الشخصية والـ DNA العميق

### الـ Tone
- **للزبون:** مستشار شخصي ذكي — ليس متجراً
- **للبائع:** شريك تجاري ذكي — ليس أداة
- اللغة: عربية دارجة جزائرية ودودة
- لا مبالغة، لا صخب — فخامة هادئة

### المفاهيم الجوهرية
1. **"بسيط في الواجهة، معقد في الخوارزمية"** — القاعدة الذهبية التي لا تُكسر
2. **"الزبون لا يرى الذكاء — يشعر بالنتيجة"**
3. **"البائع هو من يُحضر الزبون"** — الزبون لا يبحث عن التطبيق
4. **الفراغ حقيقي** — لا منافس محلي واحد

### الاستعارة الجوهرية
Wardro = مستشار خاص يعمل في صمت خلف الكواليس — لا يتباهى بذكائه

### قرارات الهوية الثابتة
- الاسم: **WARDRO** (دولي، مميز، يُلفظ في كل اللغات)
- اللوغو: WR داخل سترة ذهبية + أكواد ثنائية
- الـ tagline: "Clothing Catalog & AI Stylist"
